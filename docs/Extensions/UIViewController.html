  • openURL(_:fromSourceItem:inViewController:)

    Convenience method for opening an NSURL in a browser, giving the user a choice of opening either Google Chrome or Safari. If Safari is chosen, this calls openInSafari(_:).

    Note

    As of iOS 9, googlechrome and googlechromes should be added to the LSApplicationQueriesSchemes key in info.plist in order to open in Chrome to work.

  • paramter inViewController: The UIViewController that will present the UIAlertController. If nil, self is used to present the UIAlertController. The default value is nil.

    • -seealso: openInSafari(_:) -seealso: presentViewController(_:fromSourceItem:inViewController:animated:completion).

    Declaration

    Swift

    public func openURL(url:NSURL, fromSourceItem item:UIPopoverSourceType, inViewController:UIViewController? = nil)

    Parameters

    url

    The NSURL to open in a browser.
    item

    Either the UIView or UIBarButtonItem that the ActionSheet style UIAlertController will be presented from on Regular-Regular size class devices.
